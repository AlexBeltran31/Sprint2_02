TIENDA MySQL Exercise
This project contains a set of SQL queries designed to practice essential database skills using the Tienda schema.
The database includes two tables:
-producto (codigo, nombre, precio, codigo_fabricante)
-fabricante (codigo, nombre)
The relationship between both tables is 1:N.

Skills showed in this exercise:
-Basic SELECT statements
-Filtering, ordering, and limiting results
-String and numeric functions
-JOIN operations
-Subqyeries and nested filters
-Working with DISTINCT and aggregates
