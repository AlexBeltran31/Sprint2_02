UNIVERSIDAD MySQL Exercise
This project includes SQL practice tasks based on the Universidad database. This schema models an academic enviroment with students, professors, subjects, departments and enrolments.
Skills showed in this practice:
-Multi-table queries with INNER JOIN, LEFT JOIN and RIGHT JOIN
-Filtering, sorting and searching text patterns
-Working with dates and conditions
-Aggregations with COUNT, SUM and GROUP BY
-Subqueries and correlated queries
-Handling missing relationships (NULL values)
